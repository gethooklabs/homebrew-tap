# GetHook Homebrew Tap

Official Homebrew tap for [GetHook CLI](https://github.com/gethook/gethook-cli).

## Install

```bash
brew install gethook/tap/gethook
```

Or tap manually:

```bash
brew tap gethook/tap
brew install gethook
```

## What is GetHook CLI?

Receive, debug, and replay webhooks locally — in seconds.

```bash
gethook listen --forward-to http://localhost:3000/webhooks
```

See [gethook/gethook-cli](https://github.com/gethook/gethook-cli) for full documentation.
